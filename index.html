Compermisos PEAR

Registering the channel: 
pear channel-discover compermisos.github.com/pear
Listing available packages: 
pear remote-list -c coPear
Installing a package: 
pear install coPear/package_name
Installing a specific version/stability: 
pear install coPear/package_name-1.0.0
pear install coPear/package_name-beta
Receiving updates via a feed: 
http://compermisos.github.com/pear/feed.xml
